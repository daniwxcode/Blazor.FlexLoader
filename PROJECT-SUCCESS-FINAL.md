## ?? Fonctionnalités du composant

### ?? LoaderService - API simple et avancée
```csharp
// API simple
LoaderService.Show();
LoaderService.Close();

// API avancée  
LoaderService.Increment(); // Compteur +1
LoaderService.Decrement(); // Compteur -1
LoaderService.Reset();     // Force fermeture
```

### ?? FlexLoader Component - Hautement configurable
- **Images personnalisées** (GIF, SVG, PNG)
- **Texte personnalisable** multilingue
- **Contenu custom** avec RenderFragment
- **Styles** et couleurs configurables
- **Positionnement** flexible
- **Fermeture automatique** ou manuelle

### ?? Configuration facile
```csharp
// Program.cs
builder.Services.AddBlazorFlexLoader();

// Layout
<FlexLoader />
```
